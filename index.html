História que não pode deixar de falar

(Voltar)

HTML foi criado em 1991, por Tim Berners-Lee, no CERN (European Council for Nuclear Research) na suíça. Inicialmente oo o HTML foi projetado para interligar instituições de pesquisa próximas, e compartilhar documentos com facilidade. Em 1992, foi liberada a biblioteca de desenvolvimento WWW ( World Wide Web), uma rede de alcance mundial, que junto com o HTML proporcionou o uso em escala mundial da WEB.

Como funciona o a html é algumas tags que eu aprendi

(Voltar)

HTML é uma linguagem de marcação. Estas linguagens são constituídas de códigos que delimitam conteúdos específicos, segundo uma sintaxe própria. O HTML tem códigos para criar paginas na web. Estes códigos que definem o tipo de letra, qual o tamanho, cor, espaçamento, e vários outros aspectos do site. No início era muito complicado aprender HTML, pois eram muitos comandos para fazer algo simples. A cada nova versão, o HTML fica mais fácil de utilizar, e adquire mais funções. Atualmente qualquer pessoa pode acessar a internet a aprender a construir um site básico em questão de horas, seguindo os passos de tutoriais e aprendendo as funções de cada código.

O HTML foi a primeira linguagem de nível mundial, porem não é a única. Existem muitas outras linguagens destinadas á criação de paginas da web, porém o HTML ainda prevalece. Atualmente já é possível integrar varias linguagens na mesma pagina da Web, sendo possível usar duas ou mais linguagens no mesmo site.

Para criar e editar códigos em HTML é necessário qualquer editor de texto comum, como bloco de notas. Para testar os códigos, basta salvar o arquivo em formato .HTML e executar. Para o teste é necessário ter um navegador configurado como padrão. Não é necessária internet, pois o arquivo com os códigos esta na maquina onde esta sendo executado.

Aqui vai algumas tags que eu aprendi durante o curso de html:

(Voltar)
  1. title a onde você coloca o título que vai aparecer na aba.
  2. h1 sendo o mais importante e o título principal. Ela vai de h1 até h6 que são subtítulos de subtítulos
  3. br ela quebra linhas
  4. p um parágrafo
  5. hr define uma quebra temática em uma página HTML (por exemplo, uma mudança de tópico).
  6. u geralmente é exibido com um sublinhado
  7. strong á ao texto uma forte importância deixa em negrito
  8. mark um marcador de texto
  9. ol e ul é usando li as tag de lista ordenadas e não ordenadas
  10. small define um texto menor
  11. a de ancora define um hiperlink, que é usado para vincular de uma página a outra.
  12. sub define o texto subscrito fica o 2 do H2O
  13. sup deixa o texto pequeno em cima que nem H20 ,mas o 2 para cima
  14. blockquote especifica uma seção citada de outra fonte.
  15. font essa tag e usada no css para aplicar o tamanho da fonte e a cor do texto.
  16. del ela risca a palavra que você errou invés de apagar
  17. abbr tag define uma abreviação e seu atributo title para mostrar a descrição da abreviação/acrônimo ao passar o mouse sobre o elemento.
